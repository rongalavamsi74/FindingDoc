# Doctor Appointment Booking System (MERN Stack)
---



---

##  Features

###  User
- Register & Login
- View profile
- Apply to become a doctor
- Book appointments
- View appointment status
- Receive notifications

###  Doctor
- Doctor profile management
- View patient appointments
- Update availability timings

###  Admin
- Admin login
- Approve / reject doctor applications
- View all users
- Manage doctors
- System monitoring

---

##  Tech Stack

### Frontend
- React.js  
- Redux Toolkit  
- Material UI  
- Formik + Yup  

### Backend
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- JWT Authentication
